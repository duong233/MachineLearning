# TrafficSignClassification

## Setup
  -First, you need to create in your drive a folder name "ML"
  -Second, create child folder :"saved_model"
  -Then, in "save_model" folder, create 3 childs: model1, model2, final_model
  -Next, unzip dataset and upload it with .ipynb file to ML folder
  -Finally, open .ipnyb file to start training model

Here is the dataset link: https://www.google.com/url?q=https://d17h27t6h515a5.cloudfront.net/topher/2017/February/5898cd6f_traffic-signs-data/traffic-signs-data.zip&sa=D&source=docs&ust=1659664266185557&usg=AOvVaw1vlsluaBQO3EKb1EYxE7Lb
